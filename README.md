### REST API server for university schedule

Used by
 * Adonis js ^5.0
 * TypeScript ^4.0
 * Lodash
 * MySQL ^7.0
 
Starting
```javascript
1. node ace migration:run
2. node ace db:seed
3. npm run dev --watch
```
