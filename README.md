### REST API server for university schedule

Used by
 * Adonis js ^5.0
 * TypeScript ^4.0
 * Lodash
 * MySQL ^7.0
 
How to start?
```bash
# install dependencies
$ npm run install

# run migration
$ node ace migration:run

# run seed
$ node ace db:seed

# start local server
$ npm run dev --watch
```
